Xbotics O6 控制台 使用说明
===========================

一、快速开始
------------
1. 双击运行「Xbotics_O6控制台.exe」
2. 连接 O6 灵巧手与 CAN 转 USB 接口
3. 选择摄像头并点击「启动摄像头」
4. 在「摄像头跟随」页面进行标定后即可使用

二、主要功能
------------
【摄像头跟随】
- 将摄像头前的实时手部动作映射到 O6 灵巧手关节角度
- 首次使用需进行标定（张开→握拳→拇指内收）
- 标定完成后可持续使用，无需每次重复

【猜拳互动】
- 做出石头/剪刀/布手势，O6 会自动出克制你的手势
- 出拳后需点击「开始猜拳」重新开始下一局

【手势控制】
- 预设 15 种固定手势姿势
- 点击对应按钮即可让 O6 执行对应动作

三、标定步骤（摄像头跟随）
--------------------------
1. 点击「启动摄像头」
2. 点击「启动跟随」
3. 依次点击：
   - 「张开标定」→ 保持手掌张开约1秒
   - 「握拳标定」→ 保持握拳约1秒
   - 「拇指内收标定」→ 保持拇指向掌心收拢约1秒
4. 状态显示「已标定」后即可正常使用

四、OpenClaw 调用
-----------------
如需将 O6 交给 OpenClaw 控制：
1. 点击「OpenClaw 调用说明」
2. 复制提示词全文
3. 发送给 OpenClaw

五、常见问题
------------
- 摄像头打不开：检查设备是否被其他程序占用
- O6 未连接：检查供电、CAN 接口、runtime/config.json 配置
- 手势识别不准：重新进行标定

六、注意事项
------------
- 请保持摄像头对着手部，光线充足
- 标定后请勿大幅调整摄像头角度
- 猜拳时手势需保持稳定约1秒

技术问题请联系开发团队。
